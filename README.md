# Student-Performance-Prediction-System

The Student Performance Prediction System is a machine learning-based project that analyzes student data to predict academic performance. It helps identify strengths and weaknesses, enabling better learning strategies and academic improvement.


🚀 Features
- Predicts student performance (Good / Average / Poor)
- User-friendly interface using Tkinter
- Subject-wise insights and analysis
- Suggests learning resources for improvement


🛠️ Technologies Used
- Python
- Scikit-learn
- Pandas, NumPy
- Matplotlib
- Tkinter


📂 How to Run
- Clone the repository
- Install required libraries (pip install -r requirements.txt)
- Run the main Python file

💡 Future Scope
- Web-based version
- Database integration
- Advanced ML models and dashboards
